2026 年 4 月 27 日,le CPO de GitHub Mario Rodriguez a publié une annonce au titre discret mais au contenu qui secoue toute l'industrie : **GitHub Copilot passera à la facturation à l'usage le 1er juin 2026**. L'unité Premium Request est officiellement remplacée par les GitHub AI Credits — calculés selon les taux API publiés de chaque modèle pour les tokens (entrée, sortie, cache).

## Pourquoi GitHub doit changer

Mario Rodriguez a écrit dans l'annonce : « Aujourd'hui, une question de chat rapide et une session de codage autonome de plusieurs heures coûtent le même montant à l'utilisateur. GitHub a absorbé une grande partie de l'escalade des coûts d'inférence, mais le modèle premium request actuel n'est plus viable. »

La transformation agentique de Copilot depuis fin 2024 a multiplié le coût d'inférence par utilisateur par 5-10. La PRU ne pouvait pas distinguer « question de chat consommant 200 tokens » de « Ralph Loop de 3 heures consommant 2 millions de tokens ».

## Règles de tarification

**Toujours gratuit** : complétion de code et suggestions Next Edit (tous les plans payants)

**Consomme des Credits** : conversation, tâches agent autonomes, changement de modèle, extension de contexte. Opus 4.6 vs Sonnet 4.5 sur la même tâche = 5-8× de différence

**Code Review consomme aussi des Actions minutes** : coût additionnel
