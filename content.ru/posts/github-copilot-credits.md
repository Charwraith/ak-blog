27 апреля 2026 года CPO GitHub Марио Родригес опубликовал объявление с неприметным заголовком, но с содержанием, которое потрясло всю индустрию : **GitHub Copilot переходит на оплату по использованию 1 июня 2026 года**. Единица Premium Request официально заменяется на GitHub AI Credits — рассчитываемые по опубликованным тарифам API каждой модели для токенов (вход, выход, кеш).

## Почему GitHub должен измениться

Марио Родригес написал в объявлении : « Сегодня быстрый вопрос в чате и многочасовая автономная сессия кодирования стоят пользователю одинаково. GitHub поглотил большую часть роста затрат на инференс, но текущая модель premium request больше не жизнеспособна. »

Агентная трансформация Copilot с конца 2024 года увеличила стоимость инференса на пользователя в 5-10 раз. PRU не мог отличить « вопрос в чате, потребляющий 200 токенов » от « Ralph Loop за 3 часа, потребляющий 2 миллиона токенов ».

## Правила тарификации

**Всегда бесплатно** : автодополнение кода и предложения Next Edit (все платные планы)

**Потребляет Credits** : разговор, автономные задачи агента, переключение модели, расширение контекста. Opus 4.6 vs Sonnet 4.5 на той же задаче = 5-8× разницы

**Code Review также потребляет Actions minutes** : дополнительная стоимость
